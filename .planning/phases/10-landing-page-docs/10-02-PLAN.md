- zalopay_query_order: Check the status of a ZaloPay order by its app_trans_id.
- zalopay_refund: Refund a ZaloPay transaction by its zp_trans_id.
- zalopay_validate_callback: Validate a ZaloPay callback payload by verifying its HMAC-SHA256 MAC.

From servers/mcp-vnpay/CLAUDE.md:
- vnpay_create_payment_url: Build a signed VNPAY payment URL.
- vnpay_verify_return: Verify a VNPAY return URL by validating the HMAC-SHA512 secure hash.
- vnpay_query_transaction: Query the status of a VNPAY transaction by reference number.

From servers/mcp-zalo-oa/CLAUDE.md:
- zalo_oa_send_message: Send a message to a Zalo OA follower (text, image, file).
- zalo_oa_get_follower_profile: Get profile information for a Zalo OA follower.
- zalo_oa_list_followers: List followers of the Zalo Official Account.
- zalo_oa_refresh_token: Refresh the Zalo OA access token.
