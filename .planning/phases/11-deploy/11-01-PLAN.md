  <action>
Deploy the built dashboard to Cloudflare Pages using `wrangler pages`:

1. Create the Pages project (if it does not already exist):
```bash
npx wrangler pages project create vn-mcp-dashboard --production-branch main 2>/dev/null || true
```

2. Deploy the dist directory:
```bash
npx wrangler pages deploy apps/dashboard/dist --project-name vn-mcp-dashboard
```

The `_redirects` file in `dist/` handles SPA routing (all paths -> index.html).

The dashboard URL will be: `https://vn-mcp-dashboard.pages.dev`

If `wrangler` requires authentication, this becomes a checkpoint — the user must run `npx wrangler login` first. But since the gateway is already deployed on this machine, wrangler auth should already be configured.

After deployment, verify the URL is accessible:
```bash
curl -s -o /dev/null -w "%{http_code}" https://vn-mcp-dashboard.pages.dev
```
Expected: 200 (the SPA index.html is served).
