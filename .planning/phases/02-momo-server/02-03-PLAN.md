**Test: momo_create_payment returns payUrl with _mock:true (MOMO-01)**
```typescript
const result = await callTool(client, 'momo_create_payment', { amount: 150000, orderInfo: 'Test order from Nguyen Van A' });
expect(result.isError).toBeFalsy();
const data = parseResult(result);
expect(data._mock).toBe(true);
expect(data.payUrl).toMatch(/^https:\/\/test-payment\.momo\.vn\/pay\//);
expect(data.orderId).toMatch(/^MOMO_/);
expect(data.amount).toBe(150000);
```

**Test: momo_create_payment is deterministic (same input = same orderId)**
```typescript
const r1 = await callTool(client, 'momo_create_payment', { amount: 150000, orderInfo: 'Consistent test' });
const r2 = await callTool(client, 'momo_create_payment', { amount: 150000, orderInfo: 'Consistent test' });
expect(parseResult(r1).orderId).toBe(parseResult(r2).orderId);
```

**Test: momo_query_status returns matching transaction (MOMO-02)**
```typescript
const createResult = await callTool(client, 'momo_create_payment', { amount: 200000, orderInfo: 'Query test' });
const created = parseResult(createResult);
const queryResult = await callTool(client, 'momo_query_status', { orderId: created.orderId });
const queried = parseResult(queryResult);
expect(queried.orderId).toBe(created.orderId);
expect(queried._mock).toBe(true);
expect(queried.resultCode).toBe(0);
```

**Test: momo_refund returns successful mock refund (MOMO-03)**
```typescript
const result = await callTool(client, 'momo_refund', { transId: 2350000001, amount: 50000 });

expect(data._mock).toBe(true);
expect(data.resultCode).toBe(0);
expect(typeof data.transId).toBe('number');
expect(data.amount).toBe(50000);
```

**Test: momo_validate_ipn accepts correctly signed payload (MOMO-04)**
Craft an IPN payload object with all 13 signature fields + `signature` field:
```typescript
const credentials = getMomoCredentials();
const ipnPayload = {
  accessKey: credentials.accessKey,
  amount: 150000,
  extraData: '',
  message: 'Thanh cong.',
  orderId: 'MOMO_test123',
  orderInfo: 'Test IPN',
  orderType: 'momo_wallet',
  partnerCode: credentials.partnerCode,
  payType: 'qr',
  requestId: 'REQ_test123',
  responseTime: 1710000000000,
  resultCode: 0,
  transId: 2350000001,
  signature: '', // will be computed
};
ipnPayload.signature = buildIpnSignature(ipnPayload, credentials.secretKey);
const result = await callTool(client, 'momo_validate_ipn', { ipnBody: JSON.stringify(ipnPayload) });
const data = parseResult(result);
expect(data.valid).toBe(true);
expect(data.orderId).toBe('MOMO_test123');
expect(data.amount).toBe(150000);
```

**Test: momo_validate_ipn rejects tampered payload (MOMO-04)**
```typescript
const tampered = { ...ipnPayload, amount: 999999 }; // amount changed but signature unchanged
```
