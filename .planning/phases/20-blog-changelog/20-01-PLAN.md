2. **## The Problem** — Vietnamese developers integrating payment gateways (MoMo, ZaloPay, VNPAY) face fragmented SDKs, inconsistent docs, and boilerplate-heavy integrations. AI assistants like Claude can help write code, but they cannot directly call these APIs.

3. **## What is MCP?** — Brief explanation (3-4 sentences). Model Context Protocol is an open protocol by Anthropic that lets AI models interact with external tools. Think of it as "USB-C for AI" — a standard interface so Claude can call any tool without custom integration code. Link to Anthropic's MCP docs.

4. **## What VN MCP Hub Provides** — List the 5 servers with brief descriptions:

   - **ZaloPay** — Payment orders: create, query, refund, validate callbacks
   - **VNPAY** — Payment gateway: generate signed URLs, verify returns, query transactions
   - **Zalo OA** — Messaging: send messages, list followers, manage profiles
   - **ViettelPay** — Payments: create, query, refund (mock-only)

   Mention: 18 tools total, all with sandbox/mock mode, works offline.

5. **## Getting Started in 2 Minutes** — Two paths:

   **Self-hosted (npm):**
   ```bash
   npm install -g @vn-mcp/mcp-momo-vn
   ```

   ```json
   {
     "mcpServers": {
       "momo": {
         "command": "mcp-momo-vn",
         "env": { "MOMO_SANDBOX": "true" }
       }
     }
   }
   ```

   1. Sign up at https://vn-mcp-dashboard.pages.dev
   2. Create an API key
   3. Configure `.mcp.json` to point to `https://vn-mcp-gateway.nguyenduchuy27111997.workers.dev/mcp/momo`
