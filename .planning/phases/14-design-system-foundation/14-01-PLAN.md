2. Update `apps/dashboard/index.html` to:
   - Add Google Fonts preconnect and Inter + JetBrains Mono links in `<head>`
   - Add inline dark mode script BEFORE React — prevents flash of wrong theme
   - Remove the old inline `<style>` block (Tailwind handles this now)
   - Set `class="dark"` on `<html>` as default

   Final index.html:
   ```html
   <!DOCTYPE html>
   <html lang="en" class="dark">
     <head>
       <meta charset="UTF-8" />
       <meta name="viewport" content="width=device-width, initial-scale=1.0" />
       <title>VN MCP Hub</title>
       <link rel="preconnect" href="https://fonts.googleapis.com" />
       <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
       <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet" />
       <script>
         (function() {
           var theme = localStorage.getItem('theme');
           if (theme === 'light') {
             document.documentElement.classList.remove('dark');
           } else if (theme === 'dark' || !theme) {
             document.documentElement.classList.add('dark');
           } else if (theme === 'system') {
             if (!window.matchMedia('(prefers-color-scheme: dark)').matches) {
               document.documentElement.classList.remove('dark');
             }
           }
         })();
       </script>
     </head>
     <body>
       <div id="root"></div>
       <script type="module" src="/src/main.tsx"></script>
     </body>
   </html>
   ```
