## Description (for PH listing body)

Vietnamese developers building AI-powered apps with Claude Code have no native way to interact with local payment gateways (MoMo, ZaloPay, VNPAY, ViettelPay) or messaging platforms (Zalo OA). Every integration requires custom SDK wiring, authentication boilerplate, and webhook handling.

VN MCP Hub provides 5 Model Context Protocol (MCP) servers that wrap Vietnam's major fintech and messaging APIs. Add one line to your `.mcp.json`, and Claude Code can immediately create payments, check transactions, send Zalo messages — zero integration code. Available as npm packages for self-hosting or via a managed Cloudflare Workers gateway with a free tier (1,000 calls/month).

Built for Vietnamese developers using Claude Code or any MCP-compatible AI tool. Whether you're prototyping a payment checkout, building a Zalo chatbot, or integrating payment status checks into an AI workflow — VN MCP Hub gets you from zero to working in under 5 minutes.

## Maker Story (for "About" section)

I built VN MCP Hub because I wanted to use Claude Code with Vietnamese payment APIs but there were no MCP servers for the Vietnamese market. Every integration meant writing the same boilerplate — authentication, HMAC signing, webhook parsing. So I built 5 MCP servers covering MoMo, ZaloPay, VNPAY, ViettelPay, and Zalo OA, packaged them as npm modules, and added a hosted gateway so developers can get started in minutes without running their own server.
