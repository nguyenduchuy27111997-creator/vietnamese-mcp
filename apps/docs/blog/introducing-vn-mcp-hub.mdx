Today we are launching **VN MCP Hub** — the first collection of Model Context Protocol (MCP) servers purpose-built for Vietnamese payment gateways and messaging platforms. With 5 servers and 18 tools, Claude Code can now create payment links, check transaction status, issue refunds, and send Zalo OA messages without writing a single line of integration code. You install a package, add three lines to `.mcp.json`, and Claude takes care of the rest.

## The Problem

Vietnamese developers integrating payment gateways face a fragmented landscape. MoMo, ZaloPay, and VNPAY each have their own SDK, their own documentation style, and their own authentication scheme — HMAC-SHA256 here, SHA512 there, RSA signatures somewhere else. Every new integration means reading three different docs sites, implementing bespoke signature verification, and manually testing IPN/callback payloads.

AI coding assistants like Claude can write boilerplate, but they cannot *call* these APIs directly. Claude can draft your MoMo integration — but it cannot execute a test payment to verify the code actually works. Until now.

## What is MCP?

Model Context Protocol (MCP) is an open protocol created by Anthropic that lets AI models interact with external tools and data sources through a standardized interface. Think of it as "USB-C for AI" — one connector that works with any compatible tool, no custom integration code required.

When you configure an MCP server in your project, Claude Code gains the ability to call that server's tools during a conversation. Instead of writing code and running it manually, Claude can execute the action directly and report the result. Full documentation is available at [modelcontextprotocol.io](https://modelcontextprotocol.io).

## Getting Started in 2 Minutes

There are two ways to use VN MCP Hub: self-hosted via npm or through our hosted gateway.

### Self-hosted (npm)

Install the server globally:

```bash
npm install -g @vn-mcp/mcp-momo-vn
```

Add it to your project's `.mcp.json`:

```json
{
  "mcpServers": {
    "momo": {
      "command": "mcp-momo-vn",
      "env": {
        "MOMO_SANDBOX": "true"
      }
    }
  }
}
```

That's it. Restart Claude Code and you have MoMo tools available immediately. Replace `MOMO_SANDBOX: "true"` with real credentials when you're ready for production.

### Hosted Gateway

If you prefer not to manage server processes yourself:

1. Sign up at [vn-mcp-dashboard.pages.dev](https://vn-mcp-dashboard.pages.dev)
2. Create an API key for the server you need
3. Add the gateway URL to `.mcp.json`:

```json
{
  "mcpServers": {
    "momo": {
      "command": "npx",
      "args": ["mcp-remote", "https://vn-mcp-gateway.nguyenduchuy27111997.workers.dev/mcp/momo"],
      "env": {
        "API_KEY": "your-api-key-here"
      }
    }
  }
}
```

Then ask Claude:

> "Create a MoMo payment for 50,000 VND for order #1234"

Claude will call the `momo_create_payment` tool, return the payment URL, and show you the full API response — no manual API calls required.

<Note>
  All five servers are available on both self-hosted (npm) and hosted gateway paths. Mix and match as needed per project.
</Note>
