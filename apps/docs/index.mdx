# Vietnamese Payment APIs for Claude Code

Connect Claude Code to MoMo, ZaloPay, VNPAY, Zalo OA, and ViettelPay through the Model Context Protocol. One gateway URL — all five payment systems.

<CodeBlock language="json" title=".mcp.json">
{`{
  "mcpServers": {
    "vn-payments": {
      "url": "https://gateway.mcpvn.dev/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}`}
</CodeBlock>
