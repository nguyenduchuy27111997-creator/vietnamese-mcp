VN MCP provides 5 servers covering the major Vietnamese payment gateways and the Zalo messaging platform. All servers support mock/sandbox mode — set the `SANDBOX` env var to `true` for local development without real credentials.

## Server Comparison

| Server | npm Package | Tools | Tool Count | Type |

## Choosing a Server

- **MoMo** — Best choice for e-wallet payments. Highest penetration among urban Vietnamese users.
- **ZaloPay** — Strong adoption through Zalo ecosystem. Use with `zalopay_validate_callback` for secure webhook processing.
- **VNPAY** — Bank transfer and card payments. Required if users prefer ATM/debit cards or bank portals.
- **Zalo OA** — For notification workflows — order confirmations, OTP, customer messaging after payment.
- **ViettelPay** — Use only in staging environments. Real API uses SOAP+RSA; this server is a mock approximation.
